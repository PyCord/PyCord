``discord.ext.ipc`` -- Inter-Process Communication module for Pycord
====================================================================

``Pycord`` Provides a inter-process communication module for interacting with your WebServer, Mostly ``IPC`` is used for interacting with Websites to your bot,
if you are doing this it is suggested that you use a web host like `Flask` Or `Sanic`.
